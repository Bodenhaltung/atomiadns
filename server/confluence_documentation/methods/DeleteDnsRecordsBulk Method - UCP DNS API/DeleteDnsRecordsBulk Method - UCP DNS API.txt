Deletes all matching records from a list of zones. Everything except id must match for a record to be deleted.

h3.Declaration syntax
{panel}
void DeleteDnsRecordsBulk(
	string[] zones,
	[resourcerecord|resourcerecord Datatype - UCP DNS API][] records
)
{panel}

h3.Parameters

|| Parameter || Type || Description ||
|zones|string[]|an array with the name of the zones to remove records from|
|records|[resourcerecord|resourcerecord Datatype - UCP DNS API][]|the records to remove, everything except id must match for a record to be deleted|
